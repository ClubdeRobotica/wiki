== Desarrollo con Arduino 2013 ==

-Montaje electrónico

--Se utiliza una placa de desarrollo Arduino UNO Rev 3 como etapa de control a la cual se le adiciona un escudo (Shield) de motores Arduino que utiliza el integrado L298 para la etapa de potencia.
 
-Móvil
--Motores
--Ruedas

-Bateria
--Bateria de 9V. La placa Arduino tiene su fuente interna con regulacion a 5V y 3.3V

-Software
-Parametrización

== Hardware ==

Este proyecto esta basado en el circuito esquemático ya desarrollado en base al microcontrolador PIC.


Nos encontramos en distintas etapas de desarrollo:

-Prueba de las distintas etapas del circuito.

-Diseño de PCB en KiCad. Con el objetivo de generar una plataforma base para los futuros grupos de RSL que se incorporen al CdR.

-Elección de baterías.
